# Product Configuration

Product Configuration is used to set up all product-related information in **Koorri POS & Inventory System**.

Before adding sales, purchase, or stock, you should set up your products properly.

<Info>
Start with Product Categories, then add Measurement Units, Brands, and finally add Products.
</Info>

---

# Setup Order

Follow this order for product setup:

<Steps>
  <Step title="Create Product Categories">
    Add product groups like Medicine, Cosmetics, Grocery, Electronics, or Accessories.
  </Step>

  <Step title="Create Measurement Units">
    Add units like pcs, box, kg, liter, packet, carton, or pair.
  </Step>

  <Step title="Create Product Brands">
    Add product brand names if your products have brands.
  </Step>

  <Step title="Add Other Details">
    Add product type, color, shelf, model, and origin if needed.
  </Step>

  <Step title="Add Products">
    Add your actual products from the Product List page.
  </Step>

  <Step title="Generate Barcode">
    Generate product barcodes after products are created.
  </Step>
</Steps>

# 1. Product Categories

Product Categories help you organize products into groups.

Example:

| Category | Example Products |
|---|---|
| Medicine | Tablet, syrup, injection |
| Healthcare | Sanitary napkin, diaper, pregnancy test kit |
| Cosmetics | Cream, shampoo, face wash |
| Grocery | Rice, oil, sugar |
| Electronics | Mobile, charger, headphone |

---

## Add Product Category

<Steps>
  <Step title="Open Product Categories">
    From the left menu, go to **Product Configuration** and click **Product Categories**.
  </Step>

  <Step title="Click Add">
    Click the green **Add** button.
  </Step>

  <Step title="Enter Category Name">
    Type the category name.

    Example: `Healthcare & Medicine`
  </Step>

  <Step title="Select Under Category">
    If this is a sub-category, select the parent category.

    Example: `Diaper` can be under `Healthcare & Medicine`.

    If this is a main category, leave it empty.
  </Step>

  <Step title="Enter Ordering">
    Enter a number if you want to control the category order.

    Example: `1`, `2`, `3`
  </Step>

  <Step title="Save">
    Click **Save Product** to save the category.
  </Step>
</Steps>

<Info>
Use short and clear category names. This helps users find products easily.
</Info>

# 2. Measurement Units

Measurement Units are used to count product quantity.

Example:

| Unit | Meaning |
|---|---|
| pcs | Piece |
| box | Box |
| kg | Kilogram |
| liter | Liter |
| carton | Carton |
| packet | Packet |
| pair | Pair |

---

## Add Measurement Unit

<Steps>
  <Step title="Open Measurement Units">
    From the left menu, go to **Product Configuration** and click **Measurement Units**.
  </Step>

  <Step title="Click Add">
    Click the green **Add** button.
  </Step>

  <Step title="Enter Unit Name">
    Type the full unit name.

    Example: `Piece`
  </Step>

  <Step title="Enter Short Name">
    Type the short name.

    Example: `pcs`
  </Step>

  <Step title="Enter Base Unit">
    If this unit is connected with another unit, enter the base unit.

    Example: `pcs`
  </Step>

  <Step title="Enter Converted Ratio">
    Enter the conversion number if needed.

    Example: If `1 box = 12 pcs`, enter `12`.
  </Step>

  <Step title="Enter Tolerance">
    Enter tolerance if your business uses it.

    If not needed, keep it empty.
  </Step>

  <Step title="Save">
    Click **Save Product** to save the unit.
  </Step>
</Steps>

<Info>
If you do not use unit conversion, you can keep Base Unit, Converted Ratio, and Tolerance empty.
</Info>

---

# Add Product

<Steps>
  <Step title="Open Product List">
    From the left menu, go to **Product Configuration** and click **Product List**.
  </Step>

  <Step title="Click Add">
    Click the green **Add** button.
  </Step>

  <Step title="Enter Product Name">
    Type the product name.

    Example: `Savlon Hand Wash 250ml`
  </Step>

  <Step title="Select Category">
    Select the product category.

    Example: `Healthcare & Medicine`
  </Step>

  <Step title="Select Status">
    Select `Active` if you want to use this product in sales, purchase, and stock.
  </Step>

  <Step title="Add Description">
    Write a short product description if needed.
  </Step>

  <Step title="Upload Image">
    Upload a product image if needed.
  </Step>

  <Step title="Fill Product Attributes">
    Add unit, brand, type, size, shelf, price, opening stock, barcode, and other details.
  </Step>

  <Step title="Save Product">
    Click **Save Product** to save the product.
  </Step>
</Steps>

# Add Product Variant

Use **Add Variant** when one product has different sizes, colors, prices, or barcodes.

Example:

| Product | Variant |
|---|---|
| T-Shirt | Size M, Color Black |
| T-Shirt | Size L, Color Blue |

<Info>
Use variants when the same product has different size, color, stock, price, batch, or barcode.
</Info>

# Barcode Generator

Barcode Generator is used to create and print product barcodes.

Use barcode when:

- You scan products during sales
- You want faster invoice creation
- You want barcode labels on products
- You manage many products

# Common Mistakes

| Mistake | Result |
|---|---|
| Category not created first | Product cannot be organized properly |
| Wrong unit selected | Stock quantity may be wrong |
| Duplicate product name | Users may select the wrong product |
| Wrong price | Sales and profit reports may be incorrect |
| Wrong opening stock | Stock report may be incorrect |
| Product inactive | Product may not appear in sales or purchase |
| Wrong expiry date | Expired product report may be wrong |

---

# Best Practice

- Use clear product names
- Create categories before products
- Use correct units
- Avoid duplicate products
- Use barcode for many products
- Add opening stock carefully
- Keep product status Active for usable products
- Use shelf number for easier product finding
- Check price before saving
