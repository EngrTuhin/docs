# Banking

Banking module is used to manage all company bank accounts and mobile banking accounts used for financial transactions, payments, and receipts.

It helps businesses keep track of official banking details, account numbers, and maintain accurate records for reconciliation and payments.

---

# Purpose of Banking

You use this module to:

- Add and manage company bank accounts
- Add and manage mobile banking accounts (bKash, Nagad, Rocket, etc.)
- Store important banking information securely
- Use saved accounts in transactions, payments, and reports

---

# Banking Modules

You will find the following modules under **Finance → Banking**:

## 1. Bank Accounts

Create and manage traditional bank accounts.

## 2. Mobile Banking

Create and manage mobile financial service accounts.

---

# Summary

Banking module serves as a central repository for all banking and mobile money accounts used by the business. It ensures quick access to account details during transactions and improves financial record-keeping accuracy.

---

# 1. Bank Accounts

Bank Accounts is the module used to create and manage all traditional bank accounts of the company.

It allows businesses to store bank names, account holder names, and account numbers for easy reference.

<Info>
  This is the main setup page under Banking for managing bank accounts.
</Info>

---

## Purpose

You use this module to:

- Add new bank accounts
- View all saved bank accounts
- Maintain accurate banking information
- Use bank details in payments and reports

---

## Bank Accounts Overview

The Bank Accounts page displays a list of all saved bank accounts.

### Empty State

If no bank accounts exist, the page shows:

- "No bank found."
- A table with columns: Bank name, Account Name, Account Number, Action.

---

## Create Bank Account Form

This form appears after clicking **+ Add**.

---

## Form Fields

| Field          | Description                                             |
| -------------- | ------------------------------------------------------- |
| Bank Name      | Name of the bank (e.g., Sonali Bank, Dutch-Bangla Bank) |
| Account Name   | Name of the account holder                              |
| Account Number | Full account number                                     |

---

## Buttons & Controls

### On Listing Page

- **+ Add** (Green Button) → Create new bank account
- Search → Find bank accounts

### On Create Form

- Cancel
- Save

---

## How It Works

### Step 1: Open Module

Go to: **Finance → Banking → Bank Accounts**

### Step 2: Add Bank Account

1. Click **+ Add**
2. Enter Bank Name
3. Enter Account Name
4. Enter Account Number
5. Click **Save**

---

## After Saving

- The new bank account appears in the list
- It can be used in other modules for payments and transactions
- Can be edited or deleted from the Action column

---

## Summary

Bank Accounts is a simple but essential setup module for storing all company bank account details.

---

# 2. Mobile Banking

Mobile Banking module is used to create and manage mobile financial service accounts (such as bKash, Nagad, Rocket, Upay, etc.).

It helps businesses keep records of mobile banking numbers used for receiving payments or making disbursements.

<Info>
  This is the setup page under Banking for managing mobile banking accounts.
</Info>

---

## Purpose

This module is used to:

- Add mobile banking accounts
- Store account names and numbers
- Categorize different mobile banking types
- Support digital payment workflows

---

## Mobile Banking Overview

The Mobile Banking page displays all saved mobile banking accounts.

### Empty State

If no accounts exist, the page shows:

- "No mobilebank found."
- Table columns: Account Name, Account Number, Account Type, Action.

---

## Create Mobile Banking Form

This form appears after clicking **+ Add**.

---

## Form Fields

| Field          | Description                                         |
| -------------- | --------------------------------------------------- |
| Account Name   | Name of the account holder                          |
| Account Number | Mobile banking number                               |
| Account Type   | Type of mobile banking (e.g., bKash, Nagad, Rocket) |

---

## Buttons & Controls

### On Listing Page

- **+ Add** (Green Button) → Create new mobile banking account
- Search → Find accounts

### On Create Form

- Cancel
- Save

---

## How It Works

### Creating a New Mobile Banking Account

1. Go to **Finance → Banking → Mobile Banking**
2. Click **+ Add**
3. Enter Account Name
4. Enter Account Number
5. Select/Enter Account Type
6. Click **Save**

---

## After Saving

- The new mobile banking account is added to the list
- It becomes available for use in transactions and payment processes

---

## Summary

Mobile Banking is a setup module for managing digital wallet and mobile financial service accounts. It complements the traditional Bank Accounts module for complete banking management.
